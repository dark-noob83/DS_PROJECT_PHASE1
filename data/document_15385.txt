Blood work. These are pretty common tests that we run. There are many different types of blood work that we will run on cats. The most common four are FeLV/FIV testing, CBC, serum chemistry, and a total thyroid level.
Basic blood tests that are testing for the normal body functions like complete blood count (CBC) or a chemistry screen can be anywhere from $60-150. If you decide to run additional tests, such as Feline Immunodeficiency Virus/Feline Leukemia Virus (common screening for sick or at-risk cats), that will cost extra.
