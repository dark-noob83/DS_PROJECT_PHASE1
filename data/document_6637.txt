In computing, a locale is a set of parameters that defines the user's language, country and any special variant preferences that the user wants to see in their user interface. Usually a locale identifier consists of at least a language identifier and a region identifier.
A locale is a set of features that are culture-specific, which can be used by programs to be more portable internationally. In C++, locales are represented by an object of the. class.
In the Windows operating systems, a locale is a set of user preference information related to the user's language, environment and/or cultural conventions.
Locale support in Windows is provided through a number of different locale settings: the user locale, input locales, and the system locale. Taken together, these settings offer users a great deal of flexibility when setting up a machine.
A key concept for application programs is that of a program's locale. The locale is an explicit model and definition of a native-language environment. The notion of a locale is explicitly defined and included in the POSIX standard which can be accessed through http://opengroup.org.
