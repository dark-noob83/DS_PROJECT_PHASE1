File type specification: Program executable file type. appref-ms file icon: The appref-ms file extension is related to Microsoft Windows operating system. These files are some sort of shortcut used for programs based on Microsoft .NET framework. They are used by Microsoft ClickOnce program.
.APPREF-MS File Extension Information. .APPREF-MS is a file extension that belongs to Settings Files of Microsoft. Your Operating System will go through the registry, find the file association and then open .APPREF-MS file properly. .APPREF-MS file can be opened by many programs.

Before you can open a APPREF-MS file, you'll need to figure out what kind of file the APPREF-MS file extension refers to. Tip: Incorrect APPREF-MS file association errors can be a symptom of other underlying issues within your Windows operating system.
The APPREF-MS file type is primarily associated with 'Application Reference' by Microsoft Corporation. The difference between Application Reference files and shortcuts (.LNK) is that the application reference points to the original application URL and not the location of the EXE on the disk.
