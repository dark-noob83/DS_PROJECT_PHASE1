Debug support file (Power CTrace). TRC is a file extension for a debug file used by Oracle database software. TRC stands for TRaCe. TRC files contain an event log describing what transactions have occurred within the database. TRC files are typically used when diagnosing errors or for debugging the database. TRC files can be opened by most text editors.
